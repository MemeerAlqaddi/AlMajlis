# AL MAJLIS — GitHub Pages upload

Upload all files in this folder to the root of the `main` branch in:

https://github.com/MemeerAlqaddi/AlMajlis

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**
